## 💻 Como executar os projetos

Clone este repositório:

```bash
git clone https://github.com/LucasDuarteV/javascript-do-basico-ao-avan-ado-c-node.js-e-projetos.git
```

Entre na pasta:

```bash
cd javascript-do-basico-ao-avan-ado-c-node.js-e-projetos
```

Caso o projeto utilize Node.js:

```bash
npm install
```

Execute:

```bash
npm start
```

ou

```bash
npm run dev
```

(dependendo do projeto)
